# Flying Course Planner

Advanced military-style flying planner web application.

## Features

- Dynamic month generation
- Stage I & Stage II support
- Automatic weekend calculation
- Government holiday handling
- Equal bad weather distribution
- Progressive flying load
- A3 PDF export
- Installable PWA
- Offline support
- Add to Home Screen support
